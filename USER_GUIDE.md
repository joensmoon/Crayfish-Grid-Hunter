# 🦞 Crayfish Grid Hunter v1.0.0 用户使用手册

欢迎使用 Crayfish Grid Hunter！本手册将引导您如何在 OpenClaw 环境中安装、使用并理解这款专业的币安合约网格策略猎手。

---

## 🚀 第一步：安装 Skill

Crayfish Grid Hunter 专为 OpenClaw 环境设计，安装过程极其简便。

### 方式一：使用 ClawHub (推荐)

在您的 OpenClaw 终端中，直接运行以下命令即可一键安装 Skill 及其所有依赖：

```bash
clawhub install joensmoon/crayfish-grid-hunter
```

### 方式二：使用一键安装脚本

如果您更倾向于使用脚本，可以在 OpenClaw 终端中运行以下命令：

```bash
curl -s https://raw.githubusercontent.com/joensmoon/Crayfish-Grid-Hunter/main/install.sh | bash
```

这两种方式都会自动完成 Skill 的下载、依赖安装和配置，确保您可以立即开始使用。

---

## ✨ 第二步：触发策略扫描

安装完成后，您可以通过简单的自然语言指令在 OpenClaw 对话框中触发 Crayfish Grid Hunter 进行市场扫描。

### 核心触发指令

| 指令 | 描述 | 扫描模式 |
| :--- | :--- | :--- |
| `次新币网格` | 扫描上线 ≤90 天、市值 $10M-$200M 的潜力币种，寻找缩量横盘机会。 | Category A |
| `高波动套利` | 扫描 48 小时内波动剧烈 (>10%)、换手率极高的热点币种，寻找激进套利机会。 | Category B |
| `网格猎手` | 同时运行 Category A 和 Category B 两种扫描，并汇总推荐结果。 | All |

### 自定义参数 (通过自然语言)

Crayfish Grid Hunter 支持通过自然语言调整多种筛选参数。您只需在指令中说明您的需求即可。

**示例：**

- *“次新币网格，杠杆 3 倍，市值上限 1 亿”*
- *“高波动套利，筛选前 5 名，止损设为 3%”*
- *“网格猎手，次新币上线天数改为 60 天，高波动涨跌幅要 15% 以上”*

系统会智能解析您的指令，并将其转换为对应的 CLI 参数传递给核心引擎。

## 📊 第三步：解读扫描结果

Crayfish Grid Hunter 的输出结果以清晰的 ASCII 表格形式呈现，并包含详细的策略建议。

### 结果结构

每次扫描结果通常包含以下几个部分：

1.  **执行概览**：显示当前配置参数、扫描范围、数据来源和执行时间。
2.  **Category A (次新币网格) 推荐**：列出符合条件的次新币，包含当前价格、波动率、市值、网格参数（区间、格数、单格利润）、止损位和预估强平价。
3.  **Category B (高波动套利) 推荐**：列出符合条件的高波动币种，包含类似 Category A 的详细信息。
4.  **参数优化建议**：根据当前市场状态（如横盘震荡、趋势行情、高波动等），提供针对性的杠杆、网格数量和止损位调整建议。
5.  **防伪签名**：**这是最重要的部分！** 每次真实脚本执行的输出末尾都会包含一个唯一的 `CRAYFISH-XXXXXXXXXXXX` 签名。

### 如何解读关键指标

-   **网格区间 (Lower/Upper)**：建议的网格交易价格范围。`[✅ 在区间内]` 表示当前价格位于此区间内，适合入场；`[⚠️ 在区间外！]` 则表示当前价格已超出建议范围。
-   **格数 (Grids)**：建议的网格数量。通常波动越大，格数越多。
-   **单格利润 (Profit/Grid)**：每完成一格交易的预期利润百分比（已扣除手续费）。
-   **止损 (Stop-Loss)**：建议的硬止损价格。当价格跌破此位时，应考虑平仓。
-   **预估强平价 (Liq. Price)**：在当前杠杆下，预估的强制平仓价格。**请注意，这只是估算值，请以交易所实际显示为准！**
-   **💰 负资金费率**：如果某个币种带有 💰 符号，表示其资金费率为负。对于做多网格，这意味着您将获得资金费率收益。

## 🔒 第四步：验证输出真实性 (防伪签名)

Crayfish Grid Hunter 引入了独特的防伪签名机制，以确保您收到的结果是真实脚本执行的输出，而非 AI 模型自行生成的内容。

**如何验证：**

请务必检查每次扫描结果的**末尾**，是否包含类似以下格式的签名行：

```
⚠ 有效输出必含签名: CRAYFISH-5F687B383BB9 | 生成时间: 2026-03-15 11:45:57
```

-   **`CRAYFISH-XXXXXXXXXXXX`**：这是一个唯一的执行 ID，由脚本根据执行时间、版本号和结果数量等信息动态生成。AI 模型无法伪造此签名。
-   **`生成时间`**：显示脚本执行的 UTC 时间。

**如果 AI 的回复中没有此签名，或者签名格式不正确，则说明 AI 没有执行脚本，而是自行生成了内容。** 在这种情况下，请不要相信其结果，并尝试重新触发指令。
