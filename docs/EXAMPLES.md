# 丰富使用示例 (Examples)

本指南提供了在 OpenClaw 中使用 Crayfish Grid Hunter 的多种自然语言指令示例。

## 1. 基础策略触发

### 1.1 次新币网格 (Category A)
寻找近期上线且处于横盘整理期的合约。
*   **指令**: `"次新币网格"` 或 `"帮我找几个次新币做网格"`
*   **适用场景**: 市场整体震荡，寻找新币的突破前夜。

### 1.2 高波动套利 (Category B)
寻找市值适中、换手率高、波动率大的“妖币”。
*   **指令**: `"高波动套利"` 或 `"推荐几个高波动妖币网格"`
*   **适用场景**: 市场活跃，资金轮动快，利用高波动赚取网格利润。

### 1.3 全量扫描
*   **指令**: `"网格猎手"` 或 `"合约网格推荐"`
*   **适用场景**: 不确定当前市场风格，让系统自动从两个类别中选出最优的 Top 3。

## 2. 自定义参数示例

Crayfish Grid Hunter 支持强大的自然语言参数覆盖。您可以直接在对话中指定参数。

### 2.1 修改筛选阈値
*   **指令**: `"次新币网格，合约上线天数改为120天"`
    *   *效果*: 放宽到最近 120 天内上线的合约（默认 90 天），适合市场冷清时扩大候选池。
*   **指令**: `"高波动套利，市值范围放宽到1亿到50亿，换手率降低到30%"`
    *   *效果*: 扩大搜索范围，适合在市场冷清时寻找标的。

### 2.2 修改网格参数
*   **指令**: `"网格猎手，止损位设置在8%"`
    *   *效果*: 放宽止损到 8%，适合高波动行情（默认止损 5%）。
*   **指令**: `"次新币网格，单格利润要求最低1.5%"`
    *   *效果*: 减少网格数量，增加单格利润空间（默认 0.8%）。

## 3. 组合指令

您可以将多种要求组合在一条指令中：
*   **指令**: `"帮我扫描高波动套利，要求市值在3亿到8亿之间，杠杆用2倍，回测时间改为14天"`
*   **指令**: `"次新币网格，只要最近30天上的币，止损设宽一点到10%，给我排名前5的标的"`

## 4. 查看当前状态
如果您已经启动了实时监控：
*   **指令**: `"查看网格运行状态"` 或 `"网格表现如何"`
    *   *效果*: 输出当前的 PnL、填单率和风险警报。
