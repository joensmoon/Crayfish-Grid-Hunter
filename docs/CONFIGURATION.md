# 详细配置说明 (Configuration)

Crayfish Grid Hunter 的核心引擎 `grid_hunter_v5.py` 中定义了一个 `UserConfig` 数据类，包含了所有可供用户自定义的参数。

## 1. 类别 A：次新币横盘类 (Category A)

这些参数用于控制“次新币网格”的筛选逻辑。

| 参数名 | 默认值 | 描述 |
| :--- | :--- | :--- |
| `contract_recent_days` | `90` | 合约在币安上线的天数上限。默认 90 天内视为次新币。 |
| `volume_shrink_ratio` | `0.50` | 成交量萎缩比例。24小时成交量必须小于7天平均成交量的 50%。 |
| `atr_sideways_pct` | `2.0` | 真实波动幅度（ATR14）占比。小于 2% 视为横盘。 |
| `bb_width_sideways` | `5.0` | 布林带宽度占比。小于 5% 视为极窄幅震荡。 |
| `adx_sideways` | `20.0` | 平均趋向指数（ADX14）。小于 20 视为无明显趋势。 |

## 2. 类别 B：高波动套利类 (Category B)

这些参数用于控制“高波动套利”的筛选逻辑。

| 参数名 | 默认值 | 描述 |
| :--- | :--- | :--- |
| `mcap_min` | `200,000,000` | 最小市值（美元）。默认 2 亿美元。 |
| `mcap_max` | `1,000,000,000` | 最大市值（美元）。默认 10 亿美元。 |
| `turnover_min` | `0.50` | 24小时换手率下限。默认 50%（即成交量达到市值的一半）。 |
| `rv_min` | `15.0` | 年化实现波动率下限。默认 15%。 |
| `min_quote_volume` | `10,000,000` | 最小 24h USDT 交易量，确保流动性。 |

## 3. 网格参数 (Grid Parameters)

这些参数控制生成的等比网格策略的形态。

| 参数名 | 默认值 | 描述 |
| :--- | :--- | :--- |
| `leverage` | `5` | 合约杠杆倍数。建议不超过 20x。 |
| `min_grid_profit` | `0.008` | 扣除手续费后的单格最低利润率（0.8%）。如果低于此值，系统会自动减少网格数。 |
| `max_grid_profit` | `0.012` | 单格最高利润率目标（1.2%）。 |
| `stop_loss_pct` | `5.0` | 硬止损位。设置在网格下轨下方 5% 处。 |
| `max_exposure` | `0.08` | 最大账户风险敞口。建议单网格投入不超过总资金的 8%。 |

## 4. 回测与系统参数

| 参数名 | 默认值 | 描述 |
| :--- | :--- | :--- |
| `top_n` | `3` | 每次筛选返回的推荐标的数量。 |
| `max_symbols` | `200` | 扫描全市场时分析的合约数量上限（按交易量排序）。 |
| `backtest_days` | `30` | 历史回测的天数。 |
| `backtest_interval` | `"1h"` | 回测使用的 K 线级别。 |

## 5. 如何修改配置

在 OpenClaw 中，您无需修改代码，只需在对话中用自然语言提出即可：
> "帮我跑一次网格猎手，把 leverage 改为 3，top_n 改为 5，backtest_days 改为 14。"
